# インストール方法

Next Designのエクステンション開発支援コマンドラインツールをグローバルツールとしてnuget.orgに公開しています。
コマンドプロンプトから次のように実行してインストールします。

## NDExtのインストール
```
dotnet tool install --global NDExt 
```

なお、このプログラムを動作させるには .NET Core 3.1のSdkのインストールが必要です。
## 

## プロジェクトテンプレートのインストール
インストール後は下記のコマンドを実行しておきます。

```
ndext install
```

これで準備完了です。
